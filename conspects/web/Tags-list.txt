Тэги:
Типы: одиночные и парные / строчные и блочные / метатэги и тэги body

в Head:
1) html - открывает и закрывает файл (атрибут lang:"...")
2) head - заголовок и метаданные
3) body - тело, содержание файла
4) meta - метаданные, (непарный тэг) (внутри атрибут charset="UTF-8" - подключает все языки к файлу для их отображения)
5) title - название в браузере
6) <!DOCKTYPE html> - html-документ (пишется в самом начале)
7) link - подключение сторонних файлов, шрифтов, иконок (атрибуты: rel, type, href)
8) style - место для кода css в разделе Head


в Body:
Одиночные:
1) img - картинка (внутри атрибуты - src='', alt='', width)
2) input - поле ввода (атрибуты type (значения: text - поле для ввода текста, check - поле для галочки, password - поле для ввода пароля (почти как текст, но ввод скрывается) )
3) hr
4) br - перенос на следующую строку

Парные:
1) h1-h6 - заголовки от первого до шестого уровня
 Всегда дб заголовок 1-ого уровня
2) p - абзац (для текстов и подзаголовков)
3) a - ссылка (внутри атрибут - href='')

обёртки:
4) span - строчная обёртка
5) div - блочная обёртка (лучше не использовать внутри тексты)
5.1) template - заготовка для JS, которую не видно в HTML (как div, section)

6) <script src="script.js"></script> - добавление скрипта JS (лучше в конце перед </body>)
7) <script src="https://library-website.com/lib.js"></script> - подключение библиотеки из стороннего сайта. Перед своими js-файлами.

списки:
9) ul - маркированный список (контейнер для дочерних элементов списка)
10) ol - нумерованный список
10.1) dl - ??? (используется редко)
11) li - элементы списка
11.1) dt (элементы внутри dl)
11.2) dd (элементы внутри dl)

12) blockquote - цитата (атрибут cite - автор цитаты, можно ссылкой) (по умолчанию оформлена с отступами)

таблица:
12) table - таблица (контейнер для строк и ячеек)
13) tr - строка в таблице
14) td - ячейки строк (колонки)
15) th - заголовки колонок (ячейки первой строки) (table head)
15.1) thead - то же самое (заголовок)
15.2) tbody - контент таблицы
15.3) tfoot - подвал таблицы
(обычно <th> и <td> находятся внутри <tr>)
(текст, изображения и тп находятся только внутри <th> и <td>, <tr> и <table> предназначены исключительно для разметки)
(атрибуты colspan и rowspan определяют, сколько ячеек займёт ячейка с данным атрибутом в строке или колонке colspan="3")

поля ввода:
16) form - обёртка формы (атрибут action)
16.1) fieldset - обёртка для группы полей внутри формы
17) button - кнопка
18) textarea - поле для ввода многострочного текста
19) select - поле для выбора
19.1) option - вариант выбора (вложенный тэг в select)

стилизация:
20) em - курсив
21) strong - жирный
21) u - подчёркивание

медиа:
22) iframe (inline-frame, встроенная в строку рамка) - контейнер, в который помещают информацию с других сайтов (атрибуты height, width, src, allowfullscreen - разрешает полноэкранный режим, frameborder - отображает вокруг рамку с эффектом трёхмерности, type (?) )
*Если браузер не поддерживает iframe, между тэгами можно написать текст, который будет показываться вместо фрейма
** некоторые сайты запрещают показывать свои ресурсы во фреймах других сайтов

Youtube API: вставить в src https://www.youtube.com/embed/идентификатор_ролика?start=3
Здесь вопрос (?) - разделитель, start=3 - с 3-й секунды

23) video - вставка видео (атрибуты: src, autoplay, mute, loop, width, height, controls)
<video src="..." autoplay muted loop></video>
Видео размещается на сервере
Работает с форматами MP4, MOV, WebM, OGG (каждый формат следует проверять на Can I Use)
Если браузер определённый формат не поддерживает, можно записать несколько в тегах <source src="..." type="video/mp4(или др формат)"> внутри <video>

24) aodio - вставка аудио
Работает с форматами MP3, WAV, OGG. Если браузер не поддерживает формат - то же самое, что и с видео
(атрибуты: autoplay, controls, loop, muted, src, type)

Семантические:
Визульно ничем не отличаются от div, но удобны для чтения разработчиками, поисковыми машинами и слабовидящими (с помощью них программа определяет расположение структур сайта)
1) header - заголовок, шапка сайта
2) section - секция (объединяет несколько блоков). для отделения неравнозначного контента
3) nav - меню, навигация
4) main - основное содержимое
5) footer - подвал (нижние колонтитулы)
6) article - статья (если внутри несколько параграфов). для однотипного контента
7) aside - для выведения информации сбоку (новости, баннеры, вспомогательная навигация)
8) figure - вывод изображений (имеет внешние маржины)
8.1) figcaption (?) - подпись изображений
9) mark - выделение текста жёлтым цветом
10) adress - для адреса; выделен курсивом
11) time
main можно исп только 1 раз. header нельзя вкладывать в footer и наоборот. всё остальное можно использовать неоднократно.


Атрибуты:

атрибуты для любого тэга:
1) style - стиль CSS в HTML (пишется: <h1 style='color: red;'>...</h1>)
2) class - создаёт класс для css (class="class"). Несколько классов создаются через пробел (class='class1' class2')
3) id - создаёт id (используется обычно для подключения плагинов)

4) href - ссылка на сайт или картинку (сама ссылка в кавычках: href='...')
5) target - открыть сайт в новой вкладке или в текущей (значения: _blank - в новой вкладке)
6) src - ссылка на картинку
6.1) onerror = "..." - код, который исполняется, если не удаётся найти картинку
7) alt - текст вместо картинки, если картинка не загружается
8) width - ширина (картинки)
9) rel - назначение (rel='icon' (kb,jshortcut icon) rel='stylesheet')
10) type - тип файла (type='image')\
11) lang - определение языка для браузера для поиска, перевода сайта. Можно использовать в любых тэгах
12) charset - определяет кодировку страницы (="utf-8")
13) cite - автор/источник цитаты
14) colspan - сколько колонок займёт ячейка в строке
15) rowspan - сколько строк займёт ячейка в колонке
16) action - атрибут формы. Отрабатывает действие
17) allowfullscreen - разрешает полноэкранный режим в iframe
18) frameborder - отображает вокруг iframe рамку с эффектом трёхмерности. по умолч = 1, можно убрать значением 0 (рамку чаще изменяют в css через border, либо её удаляют)
19) атрибуты тэга <video>:
19.1) autoplay - видео запускается сразу (но некоторые браузеры это ограничивают)
19.2) controls - станет доступен виртуальный пульт управления с ползунком, громкостью и тп
19.3) loop - как только видео закончится, оно запустится с начала
19.4) muted - звук будет отключён
19.5) src - ссылка
19.6) type - формат видео (пишется type="video/mp4")
19.7) height - высота
19.8) width - ширина
19.9) poster - ссылка на изображение, которое пользователь видит, до того, как нажмёт на проигрывание
