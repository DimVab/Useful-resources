Виды селекторов:
Весь документ   * (универсальный селектор)
Тэги
Классы  .class
ID   #id


Вес селектора:
Тэг - 1
Класс - 10
ID - 100
<style>... {...}</style> стиль внутри html  - 1000

В приоритете также будет последний равнозначный селектор, тк код считывается последовательно

В HTML можно одному тэгу указывать несколько классов через пробел


ID. Мб только 1 на странице
Используются как правило для якорных ссылок на конкретный объект.

ID не рекомендуется использовать в качестве селекторов в CSS

Правила названий ID:
1) не начинать с цифр и спецсимволов
2) несколько слов разделять дефисом - или нижним подчёркиванием _ вместо пробела
3) подбирать простые слова из англ яз


Комбинации селекторов:
.class1.class2 - только, если тэг содержит оба селектора (и) (без пробела)
.class1, class2 - применяется ко обоим селекторам (или) (через запятую и пробел)
.class1 .class2 - class2 (дочерний) внутри class1 (родительский) (через пробел)

.class + .class - соседний селектор. свойства будут применяться к соседнему элементу (правому). это работает, если элементы находятся вплотную друг к другу (соответсвтенно, если между ними будет препятствие, свойство не сработает)

.class ~ .class - проставляет стили всем элементам относительно первого (вне зависимости от того, что будет между 
этими элементами)

.class1 > .class2 - class2 должен напрямую находиться в class1

.class1 * {...}  - все элементы внутри class1

*можно комбинированать несколько комбинаций

tag[class="class1"] - обращение по атрибутам
img[src='logo.png'] {...}
выбирают элементы, имеющие определённые значения атрибутов. Пишутся в квадратных скобках

tag[class$="ss"] - обращение, если класс заканчивается на "ss"
tag[class^="cl"] - обращение, если класс начинается с "cl"
a[href$="ru"] - все ссылки с доменом ru
tag[class*="las"] - обращение, если есть совпадение с "las"
tag[class~="class2"] - обращение, если есть класс class2
tag[class|="class"] - обращение, если есть полное совпадение class (но не class2, class1) либо фрагменты из class
