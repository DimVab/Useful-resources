1. Транспортный протокол SSH (SecureShell)
Шифруется
Исп для получения данных с удалённого компьютера или записи их на него
Конкретно используется для взаимодействия с Гитхабом или удалённым сервером

Для каждого рабочего компьютера нужно создать SSH-ключ и привязать к гит-аккаунту. Если у вас несколько гит-аккаунтов, на каждый нужна своя связка ключей

По умолчанию ключи находятся в домашней директории компьютера

ls -al ~/.ssh - получить ключи

папка дб пустая или её не должно быть
пример созданных ключей: 
id_dsa.pub

если не создавал их, то удалить


2. Генерация SSH-ключа:

1) Сгенерируйте в терминале приватный и публичный ключи. У публичного расширение .pub, у приватного расширения нет. Оба ключа сохранятся на вашем компьютере. Публичный нужен для привязки к «Гиту». В строке e-mail обязательно укажите адрес почты, которая привязана к «Гитхабу»:

ssh-keygen -t rsa -b 4096 -C "students-yandex@yandex.com"

2) нажать Enter, тобы сохранить в домашнем каталоге по умолчанию

3) Создайте пароль доступа к SSH-ключу. Его нужно вводить при каждом соединении через протокол. Поэтому запомните его или запишите:

> Enter passphrase (empty for no passphrase): [Type a passphrase]
> Enter same passphrase again: [Type passphrase again]

* можно оставить поле пустым и не вводить пароль

4) Запустите фоном команду ssh-agent. Она ищет SSH-ключ на вашем компьютере:
eval $(ssh-agent -s)

5) Привяжите приватный ключ к ssh-agent. Тогда вы сможете не вводить пароль при каждой работе с репозиторием:
ssh-add ~/.ssh/id_rsa


3. Привязка SSH-ключа к аккаунту GitHub

1) Скопируйте публичный ключ в буфер обмена:

clip < ~/.ssh/id_rsa.pub
      
Если clip не сработает, найдите скрытую папку .ssh, откройте файл id_rsa.pub в любом текстовом редакторе. Скопируйте его содержимое в буфер обмена.

2) В гитхабе: в левом меню выберите пункт ″SSH and GPG keys″:
3) В открывшейся вкладке выберите ″New SSH key or Add SSH key″:
4) В поле ″Title″ напишите заголовок. Например, ″Personal key″.
5) В поле ″key″ скопируйте ваш ключ из буфера обмена:
6) Нажмите кнопку ″Add SSH key″:
