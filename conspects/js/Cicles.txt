Циклы: 
Каждый повтор называется итерацией
Условие цикла
Тело цикла
Инкремент (i++), декремент (i--)

1. Цикл for (for i) - с известным числом повторений
Сначала идёт ключевое слово for, затем круглые скобки с условиями, затем фигурные скобки с кодом, который нужно повторять.
Условия в круглых скобках:
1) начальное значение: let i = 0;
2) конечное значение: i <= 4;
3) шаг счёта: i = i + 1  - счёт на каждой цифре (инкремент)
* i - переменная-счётчик
После каждого условия ставится ';' (кроме последнего)

Массивы хорошо подходят для использования в цикле
Пример массива в фигурных скобках:
console.log(array[i]);   (*array - имя массива)

for (let i = 1;  i < 8; i++) {
	console.log(i)
}

2. Цикл while - с неизвестным числом повторений
let num = ...;
while (num < 55) {
	console.log(num);
	num++; (или num += 1;)
}
Перебирает все варианты условия (в данном случае переменной), пока условие выполняется. Сначала проверяется условие, потом выполняется цикл

3. Цикл do
do {
	console.log(num);
	num++;
}
while (num < 55);
Тут сначала выполняется действие, затем проверяется  условие
Т.е. может сначала выполниться ложное условие, но потом оно прерывается


4. Цикл for in - перебор всех свойств объекта
for (let key in options) {
	...; 
}
Здесь задаётся переменная (Key) для обозначения каждого ключа и указывается объект (options). Таким образов for in перебирает каждый ключ 1 раз

5. Цикл for of - перебор всех свойств массива, строки, псевдомассива
(конструкция та же)


6. Директивы
break - остановка цикла
применяется, когда дальнейшее выполнение цикла бессысленно и может привести к неправильным результатам

continue - прерывание итерации цикла
не прерывает весь цикл, а только пропускает текущую итерацию
применяется, когда нужно настроить, какие итерации выполнять, а какие нет
