16. Методы и свойства строк и чисел
методы - функции внутри объекта     ( name.method(); ) или ( name.method1.method2(); )
свойства - вспомогательные значения  ( name.svoystvo;)
Все методы и свойства над каждыми типами данных есть в офиц документации 

Методы и свойства позволяют произвести какие-либо операции над числами или строками (в виде переменных), объектами

Метод не изменяет значения переменной, над которой выполняется,
а создаёт новое значение. При этом исходное значение переменной остаётся неизменным

16.2  Создание метода
const options = {
	option1: ...,
	option2: function() {
	...;
    }
}

options.option2();    - запуск метода
