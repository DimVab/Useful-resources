1. 1) JS подключается с помощью парного тэга <script></stript> с атрибутом src='путь к файлу' (а не href как в css)
<script src="js/script.js"><script> 
в содержании тэга в таком случае ничего не должно быть, а данный тэг пишется в самом конце body (в конце сайта)

2) либо напрямую в js: <script>alert(1);</script> (перед закрытием body)
3) с помощью атрибутов-функций внутри тэга (<p onclick="">)

Точку с запятой в конце строки не обязательно ставить, но желательно


2. Изменять элементы HTML и CSS можно через JS. Делается это через специальный "ящик"(?) document. И через функцию querySelector:
 let button = document.querySelector('.button');
Принимает на вход селектор класса элемента, к которому нужно получить доступ.

3. Взаимодействие с пользователем осуществляется с помощью "слушателя событий". Когда с элементом со слушателем совершается событие, запускается функция
элемент.addEventListener('событие', функция() { } ) ;


4. Библиотека - это сторонний js-файл с другого сайта с чужими наработками. Подключается  как обычный js-файл. В каждой библиотеке есть документация и правила использования.


Строка как тип данных на базовом уровне является объектом и получает свои методы через цепочку прототипов (?)
