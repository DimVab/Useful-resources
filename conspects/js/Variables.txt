3. Для запоминания информации создаются переменные (let ... ) которым присваиваются значения ("имя переменной" = значение)
- переменная пишется латиницей без пробелов (даже если несколько слов)
- имя задаётся такое, чтобы было ясно, какие она хранит данные (sum, username, productId)
Можно сразу создать переменную и задать ей значение (присвоить значение) в одной строке:
let peremennaya = 100;
let sum = 2 + 2; (можно сделать содержимым выражение)
Затем можно изменять значение переменной:
peremennaya = 120;
peremennaya = peremennaya + 20;
peremennaya += 1 (прибавит 1 к значению)
(* переменная - это как коробка с именем, а значение - содержимое коробки)

Также используется const
const peremennaya = 1;
Отличие в том, что let - изменяемая переменная, const - неизменяемая
(но прямых констант не бывает, иногда их можно изменить: добавлять и удалять в массивы новые значения; но целиком изменить нельзя )

В ЯП советуют везде, где можно использовать const

Названия не должны начинаться с цифр. В навании могут использоваться буквы, цифры, $, _
Нельзя повторять зарезервированные слова (error, alert и тп)
Верхний и нижний регистры имеют значение

старое объявление: var name = "dima"; (она изменяемая) (не рекомендуется использовать) (переменную var можно использовать до того, как она была объявлена. Такое явление называется "хойстинг" или всплытие переменных(???) )

если объяевление переменной было в фигурных скобках {}, то использовать её вне скобок нельзя. Это не распространяется на var

можно объявить сразу две переменные одной записью (через запятую):
let a = 5,
    b = 10;
