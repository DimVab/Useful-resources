1. Хорошим тоном  считается использование const, где это возможно (вместо let)

2. В начале JS-файла рекомендуется прописывать "use strict";
Это говорит JS, что мы работаем в современном режиме, где уже нет некоторых неточностей старого JS (???)
(раньше можно было создать переменную без ключевых слов: a = 5; с use strict такое не пройдёт)

3. В функции prompt() по умолчанию в ответ присваивается тип данных "строка"

4. Консоль в VSC определяет некоторые команды как ошибки, возможно из-за того, что некоторые команды заложены только в браузере и в нём выполняются (например prompt, confirm, alert, document)

5. if (i % 2 == 1) {
        continue;
    }				выбор всех чётных чисел

if (i % 2 == 0) {
        continue;
    }				выбор всех нечётных чисел

6. Для сравнения нескольких параметоров используется оператор && (и):
if (a > 2 && a < 10)   (чтобы выполнялись все условия)
или оператор || (или)  (чтобы выполнялось одно из условий)

7. Если нажать на кнопку "отмена" в в всплывающем окне функции prompt, то получится результат "null"
Если при этом перед prompt поставить "+" (+prompt), то null поменяется на 0

8. Тк функции можно использовать (при объявлении function name() {}) до объявления в коде, можно сгруппировать все функции в конце кода, а использовать ранее.

9. Большие однородные строчки кода полезно оборачивать в функции и вызавать их, когда они необходимы

10. Вместо нескольких строк удобнее использовать одну шаблонную строку:
console.log(`После ${hours}:00 скидка в магазине ${discount}%`);

11. Вместо нескольких строк в условии можно написать короче с операторами и (&&) и или (||)
if (year % 400 === 0 || year % 4 === 0 && year % 100 !== 0)

12. Чтобы не вводили строки или другие не-числа:
 if (typeof metres !== 'number')

13. Прописываем CSS-св-ва в JS:

resetButton.setAttribute('style', 'background-color:#f1f1f1');

другой способ: resetButton.style.backgroundColor = '#f1f1f1';


14. if ( !password ) {...}
выполняется, если password = false, а для этого он дб пустым '' или null

15. Проверка на простые числа:
function isPrime(num) {
  for (let i = 2; i < num; i += 1) {
        if (num % i === 0) {
            return false;
        }
  }
        return true
}
