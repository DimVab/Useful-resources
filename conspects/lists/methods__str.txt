Свойства строк:

Методы строк:

1) .indexOf('...') - поиск символа в строке; возвращает индекс; если символов нет, вернёт -1; если символов несколько, вернёт индекс первого из них

2) .inscludes('...') - то же самое, только true/false

3) .startsWidth('...')
4) .endsWith('...') - заканчивается ли указанными символами; возвращает true/false

5) .toUpperCase()
6) .toLowerCase() - переводит строку в нижний регистр

7) .split(разделитель) - преобразование строки в массив (разделитель - символ, по которому строка разделяется на элементы массива)

8) .slice(n1, n2) - вырезать часть строки; n1 - начало (включительно), n2 - конец (не включительно); если не указать 2-й аргумент, то строка вырезается до конца; можно указать отрицательные числа (будет считаться с конца)(счёт начиенается с -1)
можно вызвать без аргументов, тогда получится такая же строка

9) .substring(n1, n2/l1) - то же самое, только не поддерживает отриц значения; чтобы вырезать левую часть, нужно в n1 написать больший порядковый номер, а в n2 - меньший: l1 - длина символов для вырезания
